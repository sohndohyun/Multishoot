# MultiShoot
windows based invader-like game using SDL2

## 싱글모드
MultishootClient 폴더 내의 VS프로젝트를 빌드하면 아래와 같은 화면이 나옵니다.
![image1](./readmeResource/image_1.png)
첫 화면에서 싱글모드를 선택하면 계속해서 내려오는 적을 상대합니다.

마지막 점수와 최대 점수를 xml로 저장합니다.
